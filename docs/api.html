API docs by Redocly

Lingput Backend API (1.0.0)

Download OpenAPI specification:

REST API for the Lingput (Comprehensible Input) language-learning app. Uses HTTP-only cookie-based JWT auth, returns consistent envelopes for success/error, and supports async jobs for story generation.
